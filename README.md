# Магазин одежды — курсовая работа 2023

Небольшой учебный проект интернет-магазина с базовым функционалом.

## Возможности
- Регистрация и вход  
- Каталог товаров  
- Корзина  
- Оформление заказа  
- Отправка уведомления на email  
- Админ-панель для управления товарами  

## Использованные технологии
Vanilla JS, Express, Handlebars, SQLite
